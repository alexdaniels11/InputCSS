InputCSS v 0.2 beta

An assemblage of CSS3 driven styling and focus state effects to be applied to forms. Apply InputCSS styles to your forms, modify the classes to fit your needs or just use the InputCSS for idea generation to create your own effects.

Download on Github

InputCSS Basic styles

These pre-styled classes are what you want to start with when creating your forms. Depending on if you are keeping all of the borders, wanting gray borders or staying classy with the simple black borders; one of these will be your starting class. If you plan on keeping all of the borders on your form (or just want a basic clean form), start with the class basic or basic-gray before adding other classes. If you want to have a one, two or three bordered form, start with the class basic-transparent before adding other classes.
<input type="text" class="basic">
<input type="text" class="basic-gray">
<input type="text" class="basic-transparent">
<input type="text" class="basic" disabled>
<input type="text" class="basic-gray" disabled>
<input type="text" class="basic-transparent" disabled>
<input type="text" class="basic" required>
<input type="text" class="basic-gray" required>
<input type="text" class="basic-transparent" required>

Form styles

With these styles you can apply any of the previously mentioned input classes to the <form> simply by adding the word form- before any of the inputCSS classes (Example: <form class="form-basic form-fcs-box-shadow"></form>). This should make styling a lot easier if all your input fields need to look the same. And, if you still would like to create an individual input field or two different than the others, you still can by adding the regular inputCSS classes to the <input>.

The class stack allows the creator of the form to stack multiple input fields top to bottom.

<form class="form-basic stack">
							<input type="text">
<input type="text">
							</form>

Form Style Usage:

<form class="form-basic-gray form-fcs-heart-beat-border form-radius-left form-radius-btm-right">
								<label>First Name</label>
<input type="text" placeholder="First Name">
								<label>Last Name</label>
<input type="text" placeholder="Last Name">
								<label>Email</label>
<input type="text" placeholder="Email">
								</form>
							
<form class="form-basic-transparent form-border-left-sm form-border-right-sm">
								<label>Sign Up</label>
								<input type="text" placeholder="First Name" class="border-top-sm border-btm-sm radius-all">
								<input type="text" placeholder="Last Name" class="border-top-sm border-btm-sm radius-all">
								<input type="email" placeholder="Email" class="border-top-sm border-btm-sm radius-all">
								<div class="stack">
									<input type="text" placeholder="Username" class="border-top-sm radius-top border-gray-btm">
									<input type="text" placeholder="Password" class="border-gray-btm border-gray-top">
									<input type="text" placeholder="Repeat Password" class="border-btm-sm radius-btm border-gray-top">
								</div>
								</form>
