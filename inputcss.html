InputCSS Basic styles

These two pre-styled classes are what you want to start with when creating your forms. Depending on if you are keeping all of the borders or not, one or the other will be your starting option. If you plan on keeping all of the borders on your form (or just want a basic clean form), start with the class basic before adding other classes. If you want to have a one, two or three bordered form, start with the class basic-transparent before adding other classes.
<input type="text" class="basic"></input>
<input type="text" class="basic-transparent"></input>
<input type="text" class="basic" disabled></input>
<input type="text" class="basic-transparent" disabled></input>
